# Screenshots

This directory contains screenshots captured after Terraform deployment
to demonstrate operational visibility and system health.

## ops-dashboard.png
CloudWatch Operations Dashboard created by Terraform.
Shows:
- ASG CPU utilization
- ASG desired vs in-service capacity
- ALB request count
- ALB target response time
- ALB 4XX / 5XX errors
- Target health (healthy vs unhealthy)

These screenshots serve as deployment proof and portfolio evidence.
